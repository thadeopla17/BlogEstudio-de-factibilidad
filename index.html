Dimensiones de la Factibilidad

Factibilidad Técnica

¿Contamos con la infraestructura y el conocimiento para construir el sistema?

Factibilidad Económica

Análisis de costo-beneficio y retorno de inversión esperado.

Factibilidad Operativa

Resumen Ejecutivo

Desglose Económico

Matriz de Riesgos

4

Plan de Acción

Cronograma preliminar de implementación.

5

Recomendación

Aprobación final para el inicio del ciclo de desarrollo.

📌 Caso Práctico

Ejemplo Genérico: Sistema de Inventarios

Proyecto: "InventoryFlow v1.0"

Contexto: Una ferretería local pierde el 15% de su mercancía por mal seguimiento manual.

Viabilidad: Es posible implementar una base de datos local y lectores de códigos de barras básicos.

Factibilidad: El desarrollo cuesta $30,000 MXN. Se estima recuperar la inversión en 6 meses al reducir las pérdidas de stock. El equipo técnico domina SQL y Java para la implementación.
